# Api NegoHub

API realizada como parte del proyecto final para Nuevas Tecnologias de Desarrollo

## Development API

To start API, run:

```bash
npm run dev
```

Once the API is running, open your browser and navigate to `http://localhost:3000/`. 

## Project develop

The project was develop with MEAN STACK: MongoDB, Express, Angular, Node.js.
